Title Screens

The Amiga versions of some of the games (as well as the MS-DOS and Archimedes versions of "Wonderland") featured colourful title screens, and some Amiga versions also featured title music. Magnetic supports these title pictures and music, the files for which can be obtained from the Magnetic Scrolls Memorial web page. The title files should have the same name as the game file, except that the picture files must end in ".png" and the music files in ".mp3". The titles appear whenever a new game is loaded into Magnetic.
