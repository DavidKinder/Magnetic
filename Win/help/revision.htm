Revision History

Magnetic 2.3.1, Windows release 16
  • Fixed a buffer over-run found by Address Sanitizer.
  • Magnetic will have a dark user interface if Windows' dark mode is enabled.
Magnetic 2.3, Windows release 15
  • Support for high DPI and changing DPI with Windows 10: if the DPI is changed (from the Display settings app, or by moving the window from one monitor to another with a different DPI) then the interpreter rescales itself appropriately.
  • The icon and toolbar have been redrawn for a more modern look.
  • The option to specify the scaling applied to the title pictures has been removed: the interpreter now works out an appropriate scaling for itself.
  • The previous sound playing logic, which used the Amp11 library for MP3 and DirectMusic for MIDI, has been replaced by simpler calls to Windows' Media Control Interface.
Magnetic 2.3, Windows release 14
  • Added support for playing MIDI music from Wonderland.
Magnetic 2.2, Windows release 13
  • Fixed a crash on closing the window when the interpreter is waiting to show more text.
Magnetic 2.2, Windows release 12
  • Further improvements to the interpreter's appearance under Windows Vista.
Magnetic 2.2, Windows release 11
  • Minor improvements to the interpreter's appearance under Windows Vista.
Magnetic 2.2, Windows release 10
  • Corrected the file dialogs to use the Windows 2000/XP look.
Magnetic 2.2, Windows release 9
  • Corrected dialog problems with large fonts.
Magnetic 2.2, Windows release 8
  • Corrected the icon registered for .mag files.
Magnetic 2.2, Windows release 7
  • The hints window can now wrap lines of text wider than the window.
  • Added an option to control whether a separate hint window is used.
  • Added support for Windows XP look and feel.
  • Converted this file to HTML Help format.
Magnetic 2.1, Windows release 6
  • Hint files are now supported.
Magnetic 2.0, Windows release 5
  • New release 2.0 of Magnetic.
  • Added an option to have the pictures appear in the main window.
  • Added an option to a set a gamma correction value for the pictures.
Magnetic 1.0, Windows release 4
  • Title music (stored in MPEG Layer 3 files) is now supported.
Magnetic 1.0, Windows release 3
  • Title pictures are now supported.
Magnetic 1.0, Windows release 2
  • Magnetic now works when started with a filename from the command line.
  • Added Microsoft Office style toolbar and menus.
  • Added an options dialog, with options to scale pictures and set the text and background colours.
  • Added a scrollback dialog.
Magnetic 1.0, Windows release 1
  • First release.
